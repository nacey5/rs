# FindMore


#### 介绍

软件设计大赛：是对校园里社团组织的活动比赛的发布，参与以及组队参加活动的报名与参与，在这个项目中，

可以查看当前组织发布的最新的活动，以及你想要参加的活动

#### 软件架构

- 后端架构：java原生+mybatis

- 前端：原生html+css+js+原生的ajax+jquery
